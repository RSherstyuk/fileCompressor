# File Compressor

A multithreaded file compressor using zlib.

## Requirements
- C++17
- CMake 3.10+
- Zlib

## Build
```bash
mkdir build && cd build
cmake ..
cmake --build .
```

Usage: ./fileProcTest <path_to_dir>
